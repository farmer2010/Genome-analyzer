# Genome analyzer
Анализатор генома для симуляции искусственной жизни "Cyber biology 2"

Для запуска необходим модуль pyperclip

## Инструкция:

В начале кода вставляем расположение сохраненного существа для анализа. Запускаем программу, в буфер обмена скопируется код полученного графа в формате dot. Вставляем код в любой визуализатор графов, я использую https://dreampuf.github.io/GraphvizOnline , и получаем результат
